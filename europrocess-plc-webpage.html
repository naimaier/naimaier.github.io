Fluid Measurement System (PLC Webpage)

Description

Developed for Europrocess to be distributed along with automation projects for it's customers' tunnel freezers that in turn are sold to the industry. EDIT

This is a custom webpage for the embedded web server of Eliwell's Free Advance PLC that displays data collected by the machine sensors.

It features a real-time chart of the machine's temperature sensor readings.

Technical Information

Developed using HTML, CSS and Javascript. It features a mobile-first design.
