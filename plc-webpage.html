PLC Webpage

Description

Developed for Tridium to be distributed along with Tridium's automation projects for it's customers' tunnel freezers that in turn are sold to the industry.

This is a custom webpage for the embedded web server of Eliwell's Free Advance PLC. It allows reading real-time reports of the data collected by the machine sensors.

Technical Information

Developed using HTML, CSS and Javascript. I had to be careful due to the limited resources of the embedded server. It features a responsive layout and was made using Tridium's design.
